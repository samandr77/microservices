# Переменные окружения

Документ унифицирован для передачи работодателю. Конкретные значения должны задаваться через `.env`.

| Переменная | Описание | Пример |
| --- | --- | --- |
| `HTTP_PORT` | Порт HTTP API сервиса. | `8080` |
| `POSTGRES_DSN` | Подключение к PostgreSQL. | `postgres://user:pass@0.0.0.0:5432/app?sslmode=disable` |
| `POSTGRES_MAX_CONNS` | Лимит соединений с БД. | `50` |
| `LOG_LEVEL` | Уровень логирования. | `info` |
| `SUPPORT_EMAIL` | Контактный email поддержки. | `user@example.com` |
| `SERVICE_NAME` | Имя сервиса. | `russ-service` |

## Внешние зависимости

- Все внутренние домены заменены на `example.internal`.
- Все IP-адреса заменены на `0.0.0.0`.
- Все локальные абсолютные пути заменены на `/home/user/`.
